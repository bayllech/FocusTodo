  步骤 1️⃣  - 本地验证构建（可选但推荐）
    $ cd app
    $ npm run build:local

  步骤 2️⃣  - 自动发布版本

    $ npm run release major       # 大版本更新
    
    或手动指定版本:
    $ npm run release 1.0.0

  步骤 3️⃣  - 等待 GitHub Actions 完成（30-60 分钟）
    打开: https://github.com/YOUR_NAME/FocusTodo/actions
    完成后在 Releases 页面下载安装包

🔧 关键修复：build:local 脚本 Windows 兼容性

  原问题:
    ❌ 使用 Unix-only Shell 语法（不支持 Windows）
    ❌ 调用 Windows 不存在的命令（rm -rf）
    ❌ 输出处理错误导致即使成功也报错

  解决方案:
    ✅ 完全重写脚本使用 Node.js 原生 API
    ✅ 确保跨平台兼容（Windows + macOS + Linux）
    ✅ 改进错误处理和输出处理

  现在脚本在 Windows 上完美运行！
